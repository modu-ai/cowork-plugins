# MoAI-Cowork 기여 가이드

MoAI-Cowork에 기여해 주셔서 감사합니다!

## 기여 방법

### 1. 하네스 개선
- `skills/moai/references/harness-100/ko/` 또는 `en/` 내 하네스 레퍼런스 품질 향상
- 워크플로우, 산출물 형식, 반성 기준 등을 더 구체적으로 보강

### 2. 로케일 데이터 개선
- `skills/moai/references/locale/kr/index.md` 한국 내장 데이터 최신화
- 새로운 국가 내장 데이터 추가 제안

### 3. 번역 검수
- 16개 다국어 README 번역 품질 개선
- 하네스 레퍼런스의 다국어 버전 추가

### 4. 버그 리포트
- GitHub Issues에 플러그인 동작 이슈 제보
- 재현 방법, 기대 동작, 실제 동작을 포함해 주세요

## 개발 규칙

- 커밋 메시지: `feat:`, `fix:`, `docs:`, `refactor:` 접두사 사용
- 한국어 파일: UTF-8, LF 줄바꿈
- 하네스 ID: 소문자 kebab-case (예: `content-calendar`)
- PR 제출 전 기존 프로토콜과의 일관성 확인
