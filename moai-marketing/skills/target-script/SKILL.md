# 타겟 스크립트 (Target Script)

## 개요

타겟 고객 맞춤 메시징 스크립트 생성 스킬입니다. 타겟 오디언스 분석, 페인포인트 도출, 핵심 메시지 생성, 채널별 변환, A/B 테스트 제안의 5단계 파이프라인을 통해 타겟 고객에게 최적화된 메시징 스크립트를 생성합니다.

단순한 카피라이팅을 넘어, 타겟 오디언스의 심리적 특성과 페인포인트를 분석하여 설득력 있는 메시징 전략을 수립합니다.

## 워크플로우

### 단계 1: 타겟 분석 (Target Audience Analysis)

타겟 고객의 인구통계학적 특성, 심리적 특성, 행동 패턴을 분석하여 명확한 Buyer Persona를 구축합니다. 타겟 고객이 가진 문제(Pain Point)와 욕구(Desire)를 식별합니다.

**출력 항목:** Buyer Persona 프로파일, 심리적 특성, 구매 여정 단계(인지 → 고려 → 구매 → 유지 → 확산), 페인포인트 식별, 욕구 식별

### 단계 2: 페인포인트 도출 (Pain Point Extraction)

타겟 고객이 겪고 있는 구체적인 페인포인트를 3-5개 식별하고, 각 페인포인트의 심각성과 긴급성을 평가합니다. 페인포인트를 해결하는 것이 타겟 고객에게 가져다줄 가치(Value)를 명확히 합니다.

**출력 항목:** 핵심 페인포인트 3-5개, 페인포인트 심각성 평가, 페인포인트 해결 시 가치, 페인포인트별 메시지 후크

### 단계 3: 핵심 메시지 생성 (Core Message Generation)

페인포인트와 가치 제안을 바탕으로 핵심 메시지를 생성합니다. AIDA 모델(Attention-Interest-Desire-Action)과 PAS 모델(Problem-Agitate-Solution)을 활용하여 설득력 있는 메시지 구조를 설계합니다.

**출력 항목:** 핵심 가치 제안, AIDA 모델 메시지, PAS 모델 메시지, 메시지 톤앤매너

### 단계 4: 채널별 변환 (Channel-Specific Adaptation)

핵심 메시지를 각 채널(이메일, SNS, 웹, 광고)의 특성에 맞게 변환합니다. 각 채널의 길이 제한, 형식, 사용자 행동 패턴을 고려하여 최적화된 스크립트를 생성합니다.

**출력 항목:** 이메일 스크립트(콜드 이메일, 뉴스레터, 트랜잭션 이메일), SNS 스크립트(Instagram, LinkedIn, Twitter/X, TikTok), 웹 스크립트(랜딩 페이지, 제품 상세페이지, 블로그 포스트), 광고 스크립트(검색 광고, 디스플레이 광고, 동영상 광고)

### 단계 5: A/B 테스트 제안 (A/B Testing Proposal)

각 채널별로 2-3개의 변형(Variant)을 제안하고, A/B 테스트를 통해 최적의 메시지를 찾을 수 있는 실험 계획을 수립합니다.

**출력 항목:** A/B 테스트 변형(헤드라인, CTA 버튼, 메시지 톤, 길이), 테스트 가설, 성공 지표(CTR, 전환율, 오픈률, 참여율), 실험 기간

## 사용 예시

**예시 1**: B2B SaaS 제품 타겟 스크립트
- 타겟 고객: 중소 제조업체 경영진 (CEO, 생산관리자, 연 매출 10-100억원)
- 제품: AI 기반 생산 최적화 SaaS (생산 비용 15% 절감)
- 캠페인 목표: 무료 체험 신청 유도 (30일 무료 체험)
- 채널: 이메일, LinkedIn, 랜딩 페이지

**예시 2**: D2C 브랜드 타겟 스크립트
- 타겟 고객: 반려동물 보유 1인 가구, 연 소득 5천만원 이상, 2030 여성
- 제품: 프리미엄 반려동물 유기농 사료 (100% 유기농, 무添加剂)
- 캠페인 목표: 첫 구매 유도 (첫 구매 20% 할인)
- 채널: Instagram, 이메일, 랜딩 페이지

## 출력 형식

Markdown 형식의 구조화된 문서로 출력됩니다:
- 타겟 오디언스 분석
- 페인포인트 도출
- 핵심 메시지
- 채널별 스크립트
- A/B 테스트 제안

## 주의사항

### 필수 입력값

다음 4가지 입력값 중 하나라도 누락되면 스크립트 품질이 현저히 저하됩니다:
1. 타겟 고객 프로파일 (인구통계학적 특성, 심리적 특성, 행동 패턴)
2. 제품/서비스 (핵심 기능, 가치 제안, 차별화 포인트)
3. 캠페인 목표 (구매, 리드 생성, 브랜드 인지도 제고 등)
4. 채널 (이메일, SNS, 웹, 광고 중 최소 1개 이상)

### AI 슬롭 후처리

생성된 타겟 스크립트는 반드시 `moai-core:ai-slop-reviewer` 스킬을 통해 후처리해야 합니다. AI 패턴(반복적 표현, 과장된 어조, 추상적 문장)을 제거하고 실전 즉시 활용 가능한 구체적이고 명확한 스크립트로 변환합니다.
