# 브랜드 아이덴티티 (Brand Identity)

## 개요

브랜드의 전략 수립부터 네이밍, 언어 아이덴티티, 비주얼 아이덴티티, 가이드라인 문서까지 체계적으로 설계합니다.

신규 브랜드 아이덴티티 전체 설계, 기존 브랜드 리브랜딩/리뉴얼, 브랜드 네이밍 단독 개발, 슬로건/태그라인 개발, 톤앤매너 가이드 설계, 비주얼 아이덴티티 설계, 브랜드 가이드라인 문서 생성을 지원합니다.

## 트리거 키워드

브랜드 만들어줘, 브랜드 아이덴티티, 브랜드 네이밍, 이름 짓기, 슬로건, 태그라인,
톤앤매너, 브랜드 가이드라인, 로고 컨셉, CI 설계, 브랜드 전략, 브랜드 포지셔닝,
리브랜딩, 브랜드 리뉴얼, 기업 브랜딩, 브랜드 스토리, 브랜드 컬러, 브랜드 보이스

## 워크플로우

### 사전 수집: 브랜드 컨텍스트 체크리스트

작업 시작 전 다음 항목을 확인합니다. 필수 항목이 누락되면 3가지 방향을 제안하여 선택을 유도합니다.

| 항목 | 필수 | 설명 |
|------|-----|------|
| 사업 영역 | 필수 | 제품/서비스 종류, 업종 |
| 타깃 고객 | 권장 | 연령, 성별, 라이프스타일, 핵심 니즈 |
| 경쟁 브랜드 | 권장 | 벤치마크할 브랜드 2~3개 |
| 브랜드 방향 | 권장 | 원하는 이미지 키워드 |
| 기존 확정 요소 | 선택 | 이미 정해진 브랜드명, 컬러, 로고 등 |

### 1단계: 시장 분석 및 경쟁사 벤치마킹

해당 업종의 시장 현황을 분석하고 경쟁 브랜드 3개 이상을 벤치마킹합니다. 경쟁사 대비 차별화 기회를 도출합니다.

### 2단계: 브랜드 전략

포지셔닝맵 설계, 타깃 페르소나 개발, 아키타입 선택, 브랜드 에센스(미션, 비전, 핵심 가치) 정의를 수행합니다.

### 3단계: 네이밍

최소 15개 네이밍 후보를 생성하고 SMILE 평가로 정량 평가합니다. 도메인/상표 사전검증 안내를 제공하고 TOP 3를 추천합니다.

### 4단계: 언어 아이덴티티

슬로건/태그라인 개발(각 5개 이상), 브랜드 스토리 작성(300자/500자/1000자 버전), 톤앤매너 가이드 설계를 수행합니다.

### 5단계: 비주얼 아이덴티티

색상 시스템 설계(60-30-10 법칙, WCAG AA 준수), 타이포그래피 선정, 로고 컨셉 제안(2개 이상)을 수행합니다.

### 6단계: 브랜드 가이드라인 문서 생성

1~5단계 산출물을 통합하고 채널별 적용 가이드, 브랜드 용어집을 포함한 가이드라인 문서를 생성합니다.

## 사용 예시

| 요청 | 실행 범위 |
|------|----------|
| "친환경 생활용품 브랜드 만들어줘. 2030 여성 타깃, 프리미엄" | 풀 파이프라인 (1~6단계 전체) |
| "펫 케어 스타트업 브랜드 네이밍해줘" | 1~3단계 (전략+네이밍) |
| "브랜드명은 '그리너리'로 정했어. 슬로건이랑 톤앤매너 만들어줘" | 4단계 (언어 아이덴티티) |
| "프랜차이즈 카페 브랜드 리뉴얼 해줘. 기존명 '모닝빈'" | 풀 파이프라인 (기존 요소 반영) |
| "테크 스타트업 비주얼 아이덴티티 설계해줘" | 2+5단계 (전략+비주얼) |
| "패션 브랜드 가이드라인 문서로 정리해줘" | 6단계 (가이드라인 생성) |

## 주의사항

- 네이밍 후보는 반드시 15개 이상 생성하고 SMILE 매트릭스로 정량 평가합니다
- 아키타입은 주(70%) + 보조(30%) 조합으로 설계합니다
- 슬로건은 한국어/영문 각 5개 이상 제시합니다
- 색상 시스템은 60-30-10 법칙을 따르며 WCAG AA 접근성을 확보합니다
- 실제 로고 파일(.ai, .svg) 제작, 인쇄물 시안, 상표 출원 대행은 이 스킬의 범위가 아닙니다
- 도메인/상표 확인이 불가능한 경우 "[확인 필요]"로 표시하고 사용자에게 직접 확인을 권고합니다
