# K-Harness Marketplace

K-Harness — 한국발 AI 하네스 플러그인 마켓플레이스입니다.

## 설치

```
/plugin marketplace add modu-ai/k-harness
/plugin install moai-cowork@k-harness
```

## 포함 플러그인

| 플러그인 | 설명 |
|---------|------|
| [moai-cowork](plugins/moai-cowork/) | 100개 자기학습 도메인 하네스 — 글로벌 프로필, 17개 언어, 웹검색 현지화 |

## 업데이트

```
/plugin marketplace update k-harness
```
