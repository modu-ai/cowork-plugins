# PPT 디자이너 (PPTX Designer)

## 개요

발표용 슬라이드를 디자인하고 생성합니다. 3가지 생성 방식을 지원합니다: (1) pptxgenjs 코드로 직접 PPTX 생성, (2) NotebookLM용 마크다운+스타일 프롬프트, (3) 마크다운 슬라이드 원고만. Pretendard 폰트 기반 한국형 디자인 시스템을 적용합니다.

## 트리거 키워드

PPT, 파워포인트, 발표자료, 슬라이드, 프레젠테이션, 보고서, 기안서, 피칭 덱

## 워크플로우

### 1단계: 생성 방식 선택

사용자에게 생성 방식을 먼저 확인합니다:

- **pptxgenjs 직접 생성** (권장): .pptx 파일 직접 생성
- **NotebookLM용 마크다운**: NotebookLM에서 슬라이드 생성
- **마크다운 원고만**: 텍스트만 작성 (직접 디자인)

### 2단계: 콘텐츠 구조화

발표 시간, 대상 청중, 핵심 메시지를 확인 후 슬라이드 목차를 구성합니다.

**표준 발표 구조**:
- 표지 (1장): 제목, 부제, 발표자
- 목차 (1장): 섹션 개요
- 본문 (N장): 핵심 내용, 데이터, 차트
- 결론 (1장): 요약, next steps
- Q&A (1장): 질의응답

### 3단계: 디자인 시스템 적용

Pretendard 폰트 + 한국형 레이아웃 시스템을 적용합니다:

**폰트 시스템**:
- 제목: Pretendard Bold 32-44pt
- 본문: Pretendard Medium 18-24pt
- 캡션: Pretendard Regular 14-16pt

**색상 팔레트**:
- 프라이머리: #0066CC (비즈니스 블루)
- 세컨더리: #666666 (중립 그레이)
- 액센트: #FF6B35 (강조 오렌지)
- 배경: #FFFFFF (화이트) / #F5F5F5 (라이트 그레이)

**레이아웃 그리드**:
- 여백: 상하좌우 40px
- 콘텐츠 영역: 16:9 비율 유지
- 행 간격: 1.5배 (제목), 1.3배 (본문)

### 4단계: 선택된 방식으로 생성

#### 방식 1: pptxgenjs 직접 생성

```javascript
import PptxGenJS from "pptxgenjs";

const pptx = new PptxGenJS();
const slide = pptx.addSlide();

// 슬라이드 디자인
slide.addText("제목", { x: 1, y: 1, fontSize: 32, fontFace: "Pretendard" });
```

#### 방식 2: NotebookLM용 마크다운

**인포그래픽 스타일 선택**:
- 색연필 손그림: 크래프트지 위 색연필/마커 손글씨
- 미니멀 비즈니스: 흰 배경, 굵은 타이포, 아이콘 중심
- 매거진 에디토리얼: 큰 사진, 오버레이 텍스트, 세리프 폰트
- 테크 다크모드: 어두운 배경, 네온 악센트, 코드 스타일

**마크다운 슬라이드 원고 생성**:
```markdown
## 슬라이드 1: 표지
제목: {프로젝트명}
부제: {한줄 설명}

## 슬라이드 2: 목차
1. {섹션1}
2. {섹션2}
...
```

**NotebookLM 스타일 프롬프트 생성**: 선택된 스타일에 맞는 프롬프트를 생성하여 사용자가 NotebookLM에 바로 붙여넣을 수 있게 합니다.

#### 방식 3: 마크다운 원고만

마크다운 슬라이드 원고만 작성하여 전달합니다. 사용자가 직접 디자인 도구에서 사용합니다.

## 사용 예시

- "3분기 사업 성과 보고서를 10슬라이드 PPT로 만들어줘"
- "스타트업 투자 피칭 덱 15슬라이드를 디자인해줘"
- "신입사원 온보딩 교육자료를 PPT로 만들어줘"
- "경쟁사 분석 슬라이드 5장을 작성해줘"
- "제품 소개 발표자료를 Pretendard 폰트로 깔끔하게 만들어줘"

## 출력 형식

### pptxgenjs 방식

- **파일 형식**: `.pptx` (PowerPoint 2007+ 호환)
- **슬라이드 크기**: 16:9 (1920×1080)
- **폰트**: Pretendard (포함됨)
- **애니메이션**: 기본 페이드인 효과

### NotebookLM 방식

- **마크다운 파일**: 슬라이드 원고
- **스타일 프롬프트**: NotebookLM 채팅에 입력할 텍스트
- **사용 방법**: 노트북에 원고 추가 → 프롬프트 입력 → 슬라이드 자동 생성

### 마크다운 원고 방식

- **파일 형식**: `.md`
- **구조**: 각 슬라이드 = ## 제목
- **활용**: Figma, Canva, PowerPoint 등에서 직접 디자인

## 주의사항

### 슬라이드 수 가이드

발표 시간에 따른 적정 슬라이드 수:
- 3분: 5-7장
- 10분: 10-15장
- 30분: 20-30장
- 60분: 40-50장

### 한 슬라이드 한 메시지

각 슬라이드는 핵심 메시지 하나만 담도록 구성합니다. 정보가 과다하면 청중의 집중력이 분산됩니다.

### 텍스트 양 제한

슬라이드당 텍스트는 50단어 이하를 권장합니다. 상세 내용은 발표자 노트에 담고 슬라이드는 핵심 키워드 중심으로 구성합니다.

### 차트 및 시각화

데이터는 차트로 시각화합니다. 텍스트로 나열된 숫자보다 시각 자료가 1000배 더 빠르게 전달됩니다.

### 이미지 저작권

이미지 사용 시 저작권에 주의합니다. 무료 이미지 사이트(Unsplash, Pixabay 등)를 활용하거나 자체 제작 이미지를 사용합니다.

## 문제 해결

| 상황 | 해결 방법 |
|------|-----------|
| 파일 생성 실패 | pptxgenjs 설치 여부 확인: `npm install pptxgenjs`. 설치 후 재시도하세요 |
| 템플릿 미설정 | 슬라이드 주제와 대략적인 목차를 알려주시면 기본 구조로 생성해 드립니다 |
| 디자인 가이드 없음 | Pretendard 폰트 기반 기본 한국형 디자인으로 작성해 드립니다 |
| 슬라이드 수 또는 내용 방향 불명확 | 발표 시간(분)과 대상 청중을 알려주시면 최적 슬라이드 수를 추천합니다 |
| NotebookLM 연동 실패 | 마크다운 원고와 스타일 프롬프트를 별도 파일로 제공합니다. 수동으로 NotebookLM에 입력하세요 |

## 관련 스킬

- `moai-office:docx-generator` - DOCX(Word) 문서 생성
- `moai-office:hwpx-writer` - 한글(HWPX) 문서 생성
