# 워드 문서 생성기 (DOCX Generator)

## 개요

Python `python-docx` 라이브러리를 기반으로 DOCX 형식의 문서를 생성합니다. 한국 공문서 양식(행정 기관 공문, 기안문), 기업 보고서, 계약서, 제안서를 지원합니다. 플레이스홀더 기반 템플릿으로 반복 문서 작업을 자동화합니다.

## 트리거 키워드

워드, docx, Word 문서, 계약서, 공문서, 보고서 생성, 문서 편집, 기안문, 제안서, 공문, 협조문

## 워크플로우

### 1단계: 문서 유형 결정

사용자 요청 분석 후 문서 유형 선택:

- **한국 공문서**: 기안문/공문/협조문 (행정안전부 공문서 작성 규정 기준)
- **기업 보고서**: 주간/월간/분기 보고서
- **계약서**: 용역/근로/NDA/MOU
- **제안서**: 사업 제안/기술 제안
- **기타**: 회의록, 품의서, 건의사항

### 2단계: 내용 수집 및 구조화

문서 유형별 필수 항목 확인:

**공문서**: 수신처, 제목, 내용(번호 매기기), 발신 기관명, 담당자, 연락처, 날짜

**계약서**: 계약 당사자, 계약 목적, 계약 범위, 납기/기간, 대금 및 지급 조건, 지적재산권, 비밀유지, 계약 해지, 분쟁 해결, 서명란

**제안서**: 표지, 목차, 사업 개요, 현황 분석, 제안 내용, 기대 효과, 추진 일정, 예산 계획, 수행 팀 소개

### 3단계: python-docx로 문서 생성

```python
from docx import Document
from docx.shared import Pt, Cm
from docx.enum.text import WD_ALIGN_PARAGRAPH

doc = Document()

# 공문서: 굴림체 12pt, A4, 여백 상하 30mm 좌우 25mm
# 기업문서: 맑은 고딕 10-12pt
# 표: 선명한 경계선, 헤더 음영
```

### 4단계: 서식 적용 및 파일 출력

- 단락 스타일 (제목 1~3, 본문, 글머리)
- 표 삽입 및 서식 (병합, 음영, 경계선)
- 이미지 삽입 (로고, 서명)
- 머리글/바닥글 (페이지 번호, 기관명)
- 플레이스홀더 일괄 치환 (`{변수명}` → 실제 값)

## 사용 예시

- "용역 계약서 DOCX를 작성해줘 (갑: A사, 을: B사, 계약금 1,000만 원)"
- "행정기관 공문 양식으로 협조 요청 공문을 만들어줘"
- "월간 팀 성과 보고서를 Word 문서로 생성해줘"
- "NDA 초안을 DOCX로 작성해줘"
- "기업 표준 보고서 템플릿을 만들어줘"

## 출력 형식

- **파일 형식**: `.docx` (Microsoft Word 2007+ 호환)
- **페이지 설정**: A4 (210mm × 297mm)
- **여백**: 상하 30mm, 좌우 25mm (공문서 표준)
- **폰트**: 굴림/바탕/맑은 고딕 12pt (공문서), 맑은 고딕 10-12pt (기업문서)
- **인코딩**: UTF-8 (한글 완벽 지원)

## 주의사항

### HWPX 대안 사용

한컴오피스 없는 환경에서 공문서가 필요한 경우 DOCX로 대체 가능합니다. 한글 변환이 필요한 경우 `hwpx-writer` 스킬을 참조하세요.

### 공문서 규정 준수

행정안전부 「공문서 작성 규정」(훈령) 기준 양식을 준수합니다. 기관별·부서별 양식 차이가 있을 수 있으므로 기준 문서를 공유해 주시면 맞춤 작성합니다.

### 폰트 및 배포

공문서 표준 폰트는 굴림/바탕/맑은 고딕입니다. 배포 시 폰트 내장 또는 PDF 변환을 권장합니다. 수신 환경에 폰트가 설치되어 있지 않으면 대체 폰트로 렌더링됩니다.

### 전자서명

디지털 서명 필드 삽입을 지원합니다. Adobe Acrobat 호환 PDF로 변환 후 전자서명을 권장합니다.

### 보안

계약서 등 민감 문서는 비밀번호 보호 설정을 권장합니다. 문서 열기 암호와 편집 암호을 각각 설정할 수 있습니다.

## 문제 해결

| 상황 | 해결 방법 |
|------|-----------|
| 파일 생성 실패 | python-docx 설치 여부 확인: `pip install python-docx`. 설치 후 재시도하세요 |
| 템플릿 미설정 | 문서 유형과 핵심 내용을 알려주시면 기본 구조로 생성해 드립니다 |
| 폰트 깨짐 | 배포 전 PDF 변환을 권장합니다. 또는 시스템에 해당 폰트가 설치되어 있는지 확인하세요 |
| 공문서 규정 상이 | 기관별·부서별 양식 차이가 있을 수 있습니다. 기준 문서를 공유해 주시면 맞춤 작성합니다 |
| 표 서식 깨짐 | 표 병합이나 음영 설정 시 python-docx 버전을 확인하세요 (v0.8.11 이상 권장) |
