MoAI Cowork 플러그인을 활용한 실전 워크플로우 트랙을 안내합니다. 각 트랙은 실제 업무 환경에서 사용되는 **종간 간 워크플로우를 엔드투엔드로 구현**하며, 사용자의 역할에 맞는 트랙을 선택하여 따라할 수 있습니다.

## 선택 가이드

아래 6가지 트랙 중에서 현재 진행하시는 업무에 가장 적합한 트랙을 선택하세요. 각 트랙은 완결된 실전 시나리오를 기반으로 하며, 관련 스킬 체인을 통해 최종 산출물을 생성합니다.

### 📊 문서 작성 트랙
- **대상**: 기획자, 컨설턴트, 기획팀
- **주요 산출물**: 사업계획서, IR 덱, 보고서
- **스킬 체인**: strategy-planner → pptx-designer → ai-slop-reviewer
- **특징**: 데이터 기반의 전략 문서 생성, 프레젠테이션 디자인, AI 검수

### 📢 마케팅 트랙
- **대상**: 마케터, 콘텐츠 크리에이터, 브랜드 매니저
- **주요 산출물**: 블로그 포스트, SNS 콘텐츠, 이메일 캠페인
- **스킬 체인**: blog → ai-slop-reviewer → (선택) nano-banana
- **특징**: 콘텐츠 생성 → AI 검수 → 이미지 생성의 완결된 파이프라인

### 📈 데이터 분석 트랙
- **대상**: 데이터 분석가, 리서치, 기획자
- **주요 산출물**: 공공데이터 분석 시각화, 엑셀 리포트
- **스킬 체인**: public-data → data-explorer → data-visualizer → xlsx-creator
- **특징**: 공공데이터 수집 → 분석 → 시각화 → 보고서 생성

### ⚖️ 법무/컴플라이언스 트랙
- **대상**: 법무팀, 컴플라이언스 담당자, 기업 컨설턴트
- **주요 산출물**: 계약서 검토, NDA, 컴플라이언스 문서
- **스킬 체인**: contract-review → docx-generator → ai-slop-reviewer
- **특징**: 법무 리스크 분석, 표준 문서 생성, 검증 프로세스

### 💰 재무 트랙
- **대상**: 재무팀, 회계사, 경영 분석가
- **주요 산출물**: 재무제표, 세무 문서, 예실 분석 리포트
- **스킬 체인**: financial-statements → xlsx-creator → ai-slop-reviewer
- **특징**: K-IFRS 기준 재무 문서 생성, 자동화된 분석 보고서

### 🚀 제품 개발 트랙
- **대상**: 제품 매니저, 개발자, UX 디자이너
- **주요 산출물**: PRD, 로드맵, UX 스펙
- **스킬 체인**: spec-writer → pptx-designer → ai-slop-reviewer
- **특징**: 제품 요구사항 명세화, 프레젠테이션, 검증 프로세스

## 트랙 활용 방법

각 트랙은 다음과 같은 구조로 구성되어 있어 체계적으로 학습할 수 있습니다:

1. **개요**: 트랙의 목적과 주요 특징
2. **스킬 체인**: 사용되는 스킬들의 연동 구조
3. **실전 튜토리얼**: 단계별 실습 예시
4. **확장 예시**: 다양한 변형 시나리오
5. **다음 단계**: 추가 학습 제안

모든 트랙은 실제 비즈니스 시나리오를 기반으로 하며, 해당 트랙을 완료하면 관련 업무를 자동화하고 생산성을 크게 향상시킬 수 있습니다.

{{< hint type="note" >}}
각 트랙의 실습 예시는 실제 비즈니스 환경에서 바로 적용 가능하도록 설계되었습니다. 필요에 따라 예시의 파라미터를 조절하여 사용하시면 됩니다.
{{< /hint >}}
