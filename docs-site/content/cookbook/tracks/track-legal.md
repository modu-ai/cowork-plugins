**법률 트랙**은 계약서 검토부터 문서 생성, AI 검수까지 완결된 법무 컴플라이언스 워크플로우를 제공합니다. 법무팀, 컴플라이언스 담당자, 기업 컨설턴트의 법무 업무를 AI로 자동화합니다.

## 트랙 개요

### 🎯 목적
- 법무 문서 작업 과정 자동화
- 리스크 분석 및 경고 시스템 구축
- 법적 규정 준수 보장

### 📊 적용 대상
- 공급 계약서 및 거래 문서
- 비밀유지계약서 (NDA)
- 컴플라이언스 보고서
- 법적 의무성 문서

### 🛠️ 사용 플러그인
- **moai-legal**: 법무 문서 검토 및 분석
- **moai-office**: 법적 문서 생성
- **moai-core**: AI 품질 검수

## 스킬 체인

```
contract-review → docx-generator → ai-slop-reviewer
```

### Phase 1: 계약서 검토 (contract-review)
**입력**: 계약서 문서, 검토 목표  
**출력**: 리스크 분석 및 검토 보고서  
**역할**: 법적 리스크 평가, 조항 분석, 개선 제안

### Phase 2: 문서 생성 (docx-generator)
**입력**: 검토 결과 및 리스크 분석  
**출력**: 법적 문서 초안  
**역할**: 표준 계약서 생성, 조항 작성, 문서 구조화

### Phase 3: AI 품질 검수 (ai-slop-reviewer)
**입력**: 법적 문서 초안  
**출력**: 검수된 최종 문서  
**역할**: 법적 정확성 검증, 전문 용어 적절성, 완성도 검토

## 실전 튜토리얼: 공급 계약서 검토 및 리스크 분석

### 시나리오
"이 공급 계약서 리뷰해줘. 리스크 포인트 정리해줘"

### 단계별 가이드

#### Step 1: 계약서 리스크 분석
{{< terminal title="claude — cowork" >}}
> # contract-review 스킬 호출
"공급 계약서 리뷰해줘
Contract Type: 공급 계약서 (품질보증/하자보증 조항 포함)
Parties: 공급자 (A사), 구매자 (B사)
Key Areas:
- 가격 조항 및 지불 조건
- 인도 시간 및 지연 책임
- 품질 표준 및 검수 절차
- 하자 보증 및 책임 한계
- 계약 해지 조항
- 준거법 및 분쟁 해결"
{{< /terminal >}}

**분석 결과**:
- 고위험 조항 식별 (하자보증 책무 과다)
- 불균형 조항 발견 (지연 책임 vs 해지 권한)
- 누락 조항 목록 (보험 요구사항 누락)
- 개선 제안 10가지
- 법적 리스크 등급 (A: 낮음, B: 보통, C: 높음)

#### Step 2: 개선된 계약서 생성
{{< terminal title="claude — cowork" >}}
> # docx-generator 스킬 호출
"리뷰 결과를 바탕으로 개선된 계약서 초안 생성해줘
Template: 공급 계약서 표준 템플릿
Modifications:
- 하자보증 기간 12개월로 조정
- 지연 책임 상한선 설정 (최대 계약금액 10%)
- 해지 조항 양측 균형 잡기
- 보험 요구사항 명시 추가
- 준거법: 대한민국 법률 적용
Format: Word 문서 (docx) with professional formatting"
{{< /terminal >}}

**생성 문서**:
- 표준 계약서 구조 적용
- 법적 용어 정확한 사용
- 조항 번호 체계화
- 서명란 및 날짜 포함
- 첨부 문서 목록

#### Step 3: 법적 문서 검수
{{< terminal title="claude — cowork" >}}
> # ai-slop-reviewer 스킬 호출
"생성된 계약서 검수해줘
Focus: Legal accuracy, terminology, completeness
Format: Word document with legal clauses
Use Case: Business contract review"
{{< /terminal >}}

**검수 항목**:
- 법적 용어 정확성 검증
- 조항 간 논리적 일관성
- 법적 효력 있는 표현 사용
- 누락 조항 최종 확인
- 상업적 합리성 검토

#### Step 4: 최종 문서 완성
검수된 내용을 반영하여 최종 계약서 완성:
- 법무팀 최종 검토용 문서
- 상업적 조정 안내서
- 리스크 관리 체크리스트
- 다음 단계 가이드라인

### 예시 프롬프트
{{< terminal title="claude — cowork" >}}
> "IT 서비스 공급 계약서 리뷰해줘
계약 종류: 월정액 기반 서비스 계약
계약 기간: 1년 (자동갱신 옵션 포함)
서비스 내용: 클라우드 인프라 관리, 기술 지원
금액: 월 1,000만원 (VAT 별도)
리스크 포인트:
  - 서비스 수준 계약(SLA) 미비
  - 계약 해지 시 전환 계획 누락
  - 지식 이전 조항 약함"
{{< /terminal >}}

## 확장 예시

### NDA 작성
{{< terminal title="claude — cowork" >}}
> "비밀유지계약서 작성해줘
당사자: A사 (제공자), B사 (수령인)
비밀 정보: 기술 정보, 영업 정보, 고객 정보
유효 기간: 계약 종료 후 3년
적용 범위: 글로벌 (한국/미국/유럽)"
{{< /terminal >}}

**NDA 요소**:
- 정의 조항 명확화
- 예외 조항 포함
- 반환 의무 명시
- 위반 시 손해 배상 조항
- 분쟁 해결 방식

### 컴플라이언스 보고서
{{< terminal title="claude — cowork" >}}
> "GDPR 준수 보고서 생성해줘
대상: EU 고객 데이터 처리
검항 목록:
  - 데이터 처리 동의 확인
  - 데이터 주권 보장
  - 데이터 이전 안전성
  - 보존 기간 준수
  - 권리 행 절차"
{{< /terminal >}}

**컴플라이언스 체크리스트**:
- 법적 요구사항 충족 여부
- 내부 정책 일관성
- 문서화 완료 상태
- 담당자 지정
- 모니터링 절차

### ⚠️ 주의사항
{{< hint type="warning" >}}
법률 문서는 AI 생성만으로 완전한 법적 효력을 갖지 않습니다. 반드시 법무 전문가의 검토를 거쳐야 하며, AI는 보조 도구로만 활용해야 합니다. 중요한 법적 문서는 반드시 법률 전문가와 상담하시기 바랍니다.
{{< /hint >}}

- AI 생성 결과는 예비 검토용으로만 활용
- 법무 전문가 최종 검토 필수
- 관련 법규 최신 정보 확인
- 사정에 맞는 맞춤화 필요
- 법적 책임 소명 체계 구축
