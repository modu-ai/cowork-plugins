# 컴퓨터 사용

> Cowork의 컴퓨터 사용(computer use) 기능은 Claude가 마우스·키보드·화면 캡처로 데스크톱 앱이나 브라우저를 직접 조작할 수 있게 합니다.

## 언제 쓰나

- 공식 API가 없는 데스크톱 앱(예: Figma 데스크톱, 엑셀 고급 기능) 조작
- 회사 내부망 웹사이트 폼 입력
- 여러 앱을 오가는 복합 워크플로우 (예: 이메일 → 스프레드시트 → PPT)

## 권한 모델

{{< hint type="warning" >}}
컴퓨터 사용은 사용자의 실제 화면과 입력을 조작합니다. 반드시 필요한 앱에만, 일시적으로 권한을 허용하세요.
{{< /hint >}}

- 앱별로 권한 요청(**read**, **click**, **full**) — 브라우저는 기본이 **read**(화면 읽기만)로 제한
- 승인은 앱 단위로 명시적으로 이루어집니다.
- 세션이 끝나면 권한은 해제됩니다.

## 안전 수칙

- 이메일·메시지 안의 낯선 링크를 컴퓨터 사용으로 클릭하지 마세요. 브라우저 MCP로 안전하게 열도록 요청하는 편이 좋습니다.
- 금융·결제 액션(매매 주문, 송금)은 사용자가 직접 수행합니다. Cowork에 맡기지 마세요.
- 작업 중 예상 못 한 화면이 뜨면 즉시 중단시킬 수 있도록, 처음에는 짧은 단위로 시켜 보세요.

## 제한 사항

- 지문·얼굴 인증 같은 생체 인증은 자동화할 수 없습니다.
- 일부 앱은 정책상 화면 캡처가 차단됩니다.
- VM 외부 실행이 필요한 경우 엔터프라이즈 정책에 따라 제한될 수 있습니다.
