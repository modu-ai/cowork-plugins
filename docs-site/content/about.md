# 이 사이트는 무엇인가요

Claude Cowork와 `modu-ai/cowork-plugins`를 한국어로 쉽게 시작하기 위한 비공식 커뮤니티 문서입니다.

## 만드는 사람

자원봉사 성격의 한국어 문서 프로젝트이며, 원문 저작권은 [Anthropic](https://www.anthropic.com)과 [modu-ai](https://github.com/modu-ai)에 있습니다. 본 사이트는 공식 문서를 번역·재구성해 진입점 역할만 수행합니다.

## 편집 원칙

- 공식 자료와 어긋나지 않는 선에서만 요약·재구성합니다.
- 각 페이지 하단에 **Sources** 섹션으로 원문 URL을 명시합니다.
- 실행 예제는 최신 `cowork-plugins` v1.5.0 기준으로 점검합니다.
- AI가 초안을 쓰더라도 마지막에 `moai-core:ai-slop-reviewer` 스킬로 AI 패턴을 걸러내고 사람 톤으로 다듬습니다.

## 기여 방법

- 오타·오류 제보: 해당 페이지 우측의 **이 페이지 수정하기** 링크 또는 [GitHub Issues](https://github.com/modu-ai/cowork-plugins/issues)
- 쿡북 추가 제안: Issue에 `cookbook` 라벨로 제안 주세요

## 면책

본 문서는 참고용입니다. 규제 워크로드(법률·의료·금융 자문 등)에서는 반드시 [Cowork 안전 사용 가이드](cowork/safety/)를 먼저 확인하세요.
