## Highlights

v2.2.0은 **마크다운 보고서를 단일 파일 HTML로 변환**하는 `moai-content:html-report` 스킬을 신규 도입하는 MINOR 릴리스입니다. Thariq Shihipar의 **"The Unreasonable Effectiveness of HTML"** 철학을 기반으로, 외부 의존성이 거의 없는(한글 웹폰트 CDN 1개만 예외) **12-25KB 자체 완결형 HTML**을 생성합니다.

6개 보고서 모드(status/incident/plan/explainer/financial/pr)와 6개 통합 테스트를 포함하며, 기존 보고서 워크플로우에 손쉽게 통합할 수 있습니다. 인쇄 친화, 6개 폰트 스택 지원, CSS 변수 8종 등 실용 기능을 모두 갖추고 있습니다.

이번 릴리스는 또한 **SEO/AGO 최적화**를 대거 완료했습니다 — GitHub repository description, homepage URL, topics 20개 등록, Discussions 활성화, README Hero 섹션 재구성, FAQ 추가, llms.txt 제공으로 LLM 학습 최적화까지. 이는 cowork-plugins를 Google 검색과 LLM 크롤러 모두에서 발견 가능하게 만드는 중요한 단계입니다.

마켓플레이스는 107 → **108 스킬**, 플러그인은 21개 그대로입니다. **Breaking change 없음** — 기존 워크플로우 그대로 동작하며, 새로운 `html-report` 스킬은 필요할 때 자연어로 호출하면 됩니다("주간 보고서 HTML로", "재무제표 HTML 변환" 등).

## What's New (추가)

### moai-content: html-report (마크다운 → 단일 파일 HTML 변환기)

**전체 경로**: `moai-content:html-report`

**한 줄 기능 요약**: 마크다운 보고서를 인라인 SVG + vanilla JS로 구성된 단일 파일 HTML로 변환. 외부 프레임워크 의존성 0, 12-25KB 초경량 산출물.

**주요 입출력 및 지원 범위**:
- **입력**: 마크다운 보고서 파일
- **출력**: 단일 파일 HTML (자체 완결형, 인라인 CSS/JS/SVG)
- **옵션**: `mode` (status/incident/plan/explainer/financial/pr), `slug`, `output_path`, `font_stack`, `dark_mode`

**6개 보고서 모드**:
- **status**: 주간 현황 / 태스크 리스트
- **incident**: 포스트모템 / 우발 대응
- **plan**: 구현 계획 / 사업 계획
- **explainer**: 기능 설명 / 개념 해설
- **financial**: 재무 보고 / 수익 동향
- **pr**: PR 서사 / 관계자 알림

**인라인 SVG + vanilla JS**:
- 12-25KB 산출물, 페이지 로딩 거의 무영향
- `<script>`, 이벤트 핸들러, 차트·테이블 인터랙션은 모두 inline
- **외부 JS/CSS 프레임워크 의존성 0**

**한글 폰트 매핑**:
- Pretendard (기본), Noto Serif KR (serif mode), Noto Sans KR (sans), 조선일보명조 (조선일보 수거), KoPubWorld 명조, JetBrains Mono (코드 블록)
- 모드별 자동 선택 또는 `font_stack` 오버라이드

**인쇄 친화**:
- `@media print` 자동 적용
- 페이지 나누기 최적화
- 하이퍼링크 유지

**CSS 변수 계약**:
- `--ivory`, `--slate`, `--clay`, `--oat`, `--olive`, `--sans`, `--serif`, `--mono` 8개
- 모드별 기본값 포함

**P1 컨슈머 4종 호환성 검증**:
- executive-summary, financial-statements, sbiz365-analyst, daily-briefing 모두 4/5 호환성 통과

**6개 템플릿** + **6개 통합 테스트** + **design-tokens.md** (CSS 변수 명세) + **fonts.md** (폰트 매핑·CDN)

**관련 링크**:
- [SKILL.md](https://github.com/modu-ai/cowork-plugins/blob/v2.2.0/moai-content/skills/html-report/SKILL.md)

