# v2.0.x 릴리스 노트

**릴리스 날짜**: 2026-05-04
**버전**: v2.0.0 (최신, MAJOR)
**업데이트 명령**: `/plugin marketplace update cowork-plugins`

## Highlights

v2.0.0은 **한국 B2B 시장 특화** 강화 릴리스입니다. `NomaDamas/k-skill` (MIT) 의 한국 특화 스킬 6종을 cowork 컨벤션에 맞춰 포팅해, 인터넷등기소 등기부등본 일괄 발급·국토부 실거래가·식약처 의약품·식품 안전·법원경매 매각공고·KRX 시세·바른한글 맞춤법 검수까지 한국 시장에서 즉시 ROI가 발생하는 도메인 특화 스킬을 한꺼번에 도입했습니다.

마켓플레이스는 100 → **106 스킬**로 늘었으며, 플러그인은 21개 그대로입니다. **MAJOR bump이지만 Breaking change는 없습니다** — 기존 스킬·체인은 그대로 동작합니다. 1.x → 2.x 표기는 한국 B2B 시장 특화 도메인 도입이 cowork 정체성의 단계적 변화를 반영한 것입니다.

## What's New (추가)

### moai-legal: iros-registry-automation (인터넷등기소 등기부등본 일괄 발급 보조)

**전체 경로**: `moai-legal:iros-registry-automation`

**한 줄 기능 요약**: 대법원 인터넷등기소(IROS)에서 법인·부동산 등기부등본을 묶음 단위로 발급할 때 안전한 작업 순서와 로컬 자동화 보조

**주요 입출력 및 지원 범위**:
- **입력**: 법인등록번호 또는 상호명 목록 (법인), 주소·동호수 목록 (부동산)
- **출력**: 발급 PDF + 종합 리포트 (Excel)
- **자동화 범위**: 장바구니 담기, 결제 후 열람·저장, 사업자번호 → 법인정보 매핑, PDF 종합 리포트 생성
- **사용자 직접**: 로그인(공동인증서·간편인증·아이디), 카드 결제 (보안 경계)
- **제약**: 법인 결제 페이지당 10건, TouchEn nxKey 사전 설치 필요

**관련 링크**:
- [SKILL.md](https://github.com/modu-ai/cowork-plugins/blob/main/moai-legal/skills/iros-registry-automation/SKILL.md)
- [온라인 문서](https://cowork.mo.ai.kr/plugins/moai-legal/)
- [원 저작자: challengekim/iros-registry-automation](https://github.com/challengekim/iros-registry-automation) (MIT)
- [대법원 인터넷등기소](https://www.iros.go.kr)

### moai-business: real-estate-search (국토부 실거래가)

**전체 경로**: `moai-business:real-estate-search`

**한 줄 기능 요약**: 국토교통부(MOLIT) 신고 데이터 기반 한국 부동산 실거래가/전월세 조회

**주요 입출력 및 지원 범위**:
- **입력**: 지역명·법정동 코드, 거래년월(YYYYMM), 자산 타입
- **출력**: 매매·전월세 실거래 목록 + 중위값/최소/최대 요약
- **지원 자산**: 아파트·오피스텔·연립다세대·단독다가구·상업업무용 (매매), 아파트·오피스텔·연립·단독 (전월세)
- **사용자 측 API 키 불필요**: NomaDamas hosted proxy(`k-skill-proxy.nomadamas.org`)가 `DATA_GO_KR_API_KEY` 보유

**관련 링크**:
- [SKILL.md](https://github.com/modu-ai/cowork-plugins/blob/main/moai-business/skills/real-estate-search/SKILL.md)

### moai-commerce: mfds-safety (식약처 의약품·식품 안전 통합)

**전체 경로**: `moai-commerce:mfds-safety`

**한 줄 기능 요약**: 식품의약품안전처(MFDS) 의약품·식품 안전 정보 통합 조회 (인터뷰 우선, red flag 응급 안내)

**주요 입출력 및 지원 범위**:
- **입력**: 약 이름, 식품명, 증상, 복용·섭취 상황
- **출력**: 효능·주의·상호작용·기능성·섭취량·부적합 사유 요약
- **데이터**: 의약품 (e약은요·안전상비의약품), 건강기능식품 (I-0040 기능성 원료 / I-0050 개별인정형 / I0030 품목제조 신고), 검사부적합 (I2620), 회수·판매중지 (I0490)
- **HARD 정책**: red flag(호흡곤란·의식저하·혈변·심한 탈수 등) 발견 시 **즉시 119·응급실 안내**가 우선

**관련 링크**:
- [SKILL.md](https://github.com/modu-ai/cowork-plugins/blob/main/moai-commerce/skills/mfds-safety/SKILL.md)
- [온라인 문서](https://cowork.mo.ai.kr/plugins/moai-commerce/)
- [식의약 데이터 포털](https://data.mfds.go.kr)

### moai-finance: court-auction-search (법원경매 매각공고)

**전체 경로**: `moai-finance:court-auction-search`

**한 줄 기능 요약**: 대법원 법원경매정보(`courtauction.go.kr`) 부동산 매각공고·사건번호 단건 조회 (read-only)

**주요 입출력 및 지원 범위**:
- **입력**: 매각기일, 법원사무소코드, 입찰구분(기일/기간), 사건번호
- **출력**: 매각공고 카드 목록, 사건/물건 상세 (사건번호·용도·주소·감정평가액·최저매각가)
- **Throttling**: 호출 간 최소 2초, 세션당 10회 budget, IP 차단 시 즉시 멈춤 (자동 retry 금지)
- **HARD 사실**: 사이트는 **공식 공개 OpenAPI를 제공하지 않으며**, 본 스킬은 사이트 내부 WebSquare JSON XHR endpoint를 직접 호출합니다. 실제 입찰 전에는 법원 원문을 재확인해야 합니다

**관련 링크**:
- [SKILL.md](https://github.com/modu-ai/cowork-plugins/blob/main/moai-finance/skills/court-auction-search/SKILL.md)

- [대법원 법원경매정보](https://www.courtauction.go.kr)
- [등기정보광장 OpenAPI (별개 서비스)](https://data.iros.go.kr)

### moai-finance: korean-stock-search (KRX 시세)

**전체 경로**: `moai-finance:korean-stock-search`

**한 줄 기능 요약**: KRX(한국거래소) 상장 종목 검색·기본정보·일별 시세 조회

**주요 입출력 및 지원 범위**:
- **입력**: 종목명·종목코드, 시장(`KOSPI`/`KOSDAQ`/`KONEX`), 기준일(`YYYYMMDD`)
- **출력**: 검색 결과·기본정보·일별 시세 (종가·등락률·거래량·시가총액)
- **데이터 갱신**: 일 1회 (영업일 +1, 13시 이후), 2010년 이후 데이터 보유, 실시간 호가·체결 미제공
- **사용자 측 `KRX_API_KEY` 발급 불필요**: NomaDamas hosted proxy 경유
- **HARD Disclaimer**: 투자 자문 아님. 답변 말미에 "KRX 공식 데이터 기준 / 투자 조언 아님" 항상 고지

**관련 링크**:
- [SKILL.md](https://github.com/modu-ai/cowork-plugins/blob/main/moai-finance/skills/korean-stock-search/SKILL.md)

## 업그레이드 방법

### 1단계: 마켓플레이스 업데이트

```bash
/plugin marketplace update cowork-plugins
```

### 2단계: 플러그인 상세 재진입

Claude Cowork에서 영향받은 5개 플러그인(`moai-legal`, `moai-business`, `moai-commerce`, `moai-finance`, `moai-content`)의 상세 페이지를 다시 열면 신규 스킬이 표시됩니다.

### 3단계: API 키 (대부분 불필요)

신규 6스킬 모두 **사용자 측 API 키 발급이 불필요**합니다. NomaDamas의 hosted proxy(`k-skill-proxy.nomadamas.org`)가 운영 측 키를 보유합니다.

self-host가 필요한 경우에만:

```bash
# self-host 프록시 사용 시
KSKILL_PROXY_BASE_URL=https://your-proxy.example.com

# self-host 프록시 운영 측에서만
DATA_GO_KR_API_KEY=...      # 공공데이터포털 (real-estate, mfds)
FOODSAFETYKOREA_API_KEY=... # 식품안전나라 (mfds 식품 부분)
KRX_API_KEY=...             # KRX Open API (korean-stock)
```

### 4단계: 호환성 확인

기존 워크플로우는 그대로 동작합니다. v1.8.x에서 작성한 SPEC·체인은 v2.0.0에서 변경 없이 실행됩니다.

## 사용 예시

### 인터넷등기소 등기부등본 일괄 발급 (실사·법무)

```
> 실사 대상 법인 30개 등기부등본 한 번에 발급해줘.
  법인등록번호 목록 corp-input.json에 정리해뒀어.
  로그인은 내가 직접 할게.
→ iros-registry-automation 스킬 실행
→ 사용자 직접 로그인 → 장바구니 자동 → 사용자 결제 → 자동 다운로드 → Excel 종합 리포트
```

### 부동산 시세 분석

```
> 강남구 아파트 2024년 3월 매매 실거래가 보여줘
→ real-estate-search 스킬 실행 (region-code → apartment/trade)
→ 중위값·최소·최대 + 상위 5건 + 출처(국토부 실거래가 신고)
```

### 헬스 커머스 상품 안전 검수

```
> 차전자피 함유 신상품 출시 검토 — 1일 섭취량과 회수 이력 확인해줘
→ mfds-safety 스킬 실행 (인터뷰 → product-report → health-food-ingredient → inspection-fail)
→ 기능성·섭취량·주의사항·부적합 사유 요약
```

### 법원경매 매각공고 검토 (자산 처분)

```
> 서울중앙지방법원 2026-04-27 기일입찰 매각공고 보여줘
→ court-auction-search 스킬 실행 (2초 throttle)
→ 매각공고 카드 → 사건번호·용도·주소·감정평가액·최저매각가
→ "참고용·실제 입찰 전 법원 원문 재확인" 고지 동반
```

### KRX 시세 + DART 공시 결합 (IR)

```
> Series A IR 덱 — 삼성전자 KRX 시세와 DART 공시 데이터 결합해줘
→ korean-stock-search (KRX 시세) + DART MCP (공시) → investor-relations → pptx-designer
```

### 한국어 텍스트 최종 검수 체인

```
> 블로그 발행 — 본문 ai-slop-reviewer 검수 후 바른한글로 맞춤법까지
→ blog → ai-slop-reviewer (AI 패턴) → korean-spell-check (띄어쓰기·맞춤법) → 사용자 최종 검토
```
