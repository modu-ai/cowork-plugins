# Growth Hacking (harness-100-growth)

> MoAI-Cowork V.0.1.3 Harness Reference

## Overview
Growth Hacking provides a rapid experimentation framework for scaling user acquisition and product adoption through viral loops, product-led growth tactics, and conversion funnel optimization. This harness empowers non-technical users to design referral programs, launch A/B tests, identify growth bottlenecks, and measure engagement metrics without requiring engineering or advanced analytics skills.

## Expert Roles

**Growth Strategy Lead**
- Define growth hypothesis and prioritize experimentation roadmap
- Map user journey from awareness through advocacy
- Identify growth loops (viral coefficient, network effects, retention triggers)
- Set ambitious yet realistic growth targets aligned to business goals
- Align cross-functional teams (product, marketing, analytics) on growth strategy

**Viral Loop Designer**
- Architect referral mechanics (incentive structures, sharing mechanisms, tracking)
- Design network effects and multi-tier referral rewards
- Create user journey touchpoints that encourage sharing
- Build viral coefficient models and predict organic growth amplification
- Document loop mechanics, edge cases, and fraud prevention measures

**Conversion Optimizer**
- Audit conversion funnel and identify drop-off points (signup, onboarding, activation)
- Design and manage A/B tests for landing pages, CTAs, pricing models
- Test messaging, imagery, and user flow variations systematically
- Calculate win/loss thresholds and statistical significance requirements
- Continuously iterate on funnel improvements based on test results

**Product-Led Growth Manager**
- Implement free trial structures, freemium tier mechanics, or demo experiences
- Design onboarding flows that drive feature adoption and time-to-value
- Create in-app notifications and engagement triggers for feature discovery
- Measure product activation rates and build predictive churn models
- Optimize user experience through behavioral analytics and heatmaps

**Analytics & Insights Specialist**
- Define and track KPIs (CAC, LTV, retention, viral coefficient, engagement)
- Build dashboards for real-time growth metrics and funnel analytics
- Conduct cohort analysis to identify high-value user segments
- Generate weekly/monthly growth reports with actionable insights
- Identify growth acceleration opportunities through data analysis

## Workflow

### Phase 1: Context Gathering
- Define core growth metric and business constraints (budget, timeline, target audience)
- Audit current user acquisition, activation, and retention metrics
- Identify competitive landscape and differentiation opportunities
- Map existing user journeys and touchpoints
- Establish baseline metrics and A/B test infrastructure

### Phase 2: Analysis & Production
- Conduct growth modeling and opportunity assessment
- Design 3-5 growth experiments with clear hypotheses and success criteria
- Launch A/B tests and viral loop initiatives
- Monitor real-time test results and interim metrics
- Gather qualitative feedback through user interviews and surveys

### Phase 3: Review & Delivery
- Analyze experiment results and calculate statistical significance
- Document learnings, winning variations, and scaling recommendations
- Implement winning variations and iterate on next experiments
- Share growth playbook with stakeholders and document standard procedures
- Plan next growth cycles with refined hypotheses

## Deliverables
- Growth Strategy Canvas (goals, target audience, key metrics, initiatives)
- Referral Program Design Document (mechanics, incentives, tracking, fraud prevention)
- Funnel Audit Report (current state, drop-off analysis, optimization opportunities)
- A/B Test Plan & Results (hypotheses, variations, sample size, statistical significance)
- Viral Loop Model (coefficient calculation, organic amplification projections)
- Weekly Growth Dashboard (KPIs, cohort analysis, funnel metrics)
- Growth Playbook (repeatable processes, experiment templates, quick wins checklist)

## Extension Skills
- Paid acquisition strategy and marketing mix optimization
- Customer lifetime value modeling and retention analytics
- Community building and user advocacy programs

## Error Handling
- **Underpowered tests**: Ensure minimum sample sizes; extend test duration before concluding; document early stopping rules
- **Metric gaming**: Define success with balanced scorecard (avoid vanity metrics); prioritize user value over raw numbers
- **Technical implementation errors**: Test viral mechanics thoroughly; validate tracking and attribution before scaling
- **Adverse user experience**: Monitor qualitative feedback and NPS alongside experiment metrics; pivot if experience degrades
- **Experiment fatigue**: Rotate user cohorts; limit concurrent tests; maintain control groups for learning
- **Seasonal effects**: Control for timing; run tests across multiple weeks/months; account for competitive or market events
