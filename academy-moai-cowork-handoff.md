# academy.mo.ai.kr — moai-cowork 소개 핸드오프 문서

> **버전**: v2.5.0 (2026-05-13) | **작성일**: 2026-05-13
> **대상**: academy.mo.ai.kr 메인 랜딩페이지에 삽입할 마크다운 콘텐츠
> **독자**: 수강 예정자 (일반인 / 셀러 / 1인 기업)
> **중심 도메인**: 모두의 커머스 3일 마스터 캠프

---

## 1. moai-cowork 한 줄 소개

**"한국 셀러를 위한 AI 업무 자동화 도구 모음"**

moai-cowork은 Claude(클로드) AI에 연결하는 플러그인 모음입니다. 설치하면 자연어 한 줄로 **상세페이지, 광고 소재, SNS 콘텐츠, 사업계획서, PPT, 영상** 등을 바로 만들 수 있습니다. 한국 이커머스 환경(쿠팡, 네이버 스마트스토어, 카페24 등)에 특화되어 있으며, 복잡한 설정 없이 대화만으로 결과물을 얻을 수 있습니다.

### 핵심 숫자

| 항목 | 수량 |
|------|:----:|
| 도메인 플러그인 | 21개 |
| 전체 스킬(기능) | 130개 |
| 커머스 캠프 연관 스킬 | 40개+ |
| 외부 의존성 | API 키 3~5개 (선택) |
| 라이선스 | MIT (무료) |

---

## 2. 커머스 캠프에서 사용하는 플러그인

3일 캠프에서는 아래 6개 플러그인을 중심으로 실습합니다.

```
moai-core (필수)     ← 모든 스킬의 출발점, 자동 라우팅
moai-commerce         ← 상세페이지·광고·마진·자동화
moai-media            ← 이미지·영상·음성 AI 생성
moai-marketing        ← SNS·캠페인·광고 심리학
moai-content          ← 랜딩페이지·카드뉴스·카피
moai-office           ← PPT·Word·Excel 문서 생성
```

## 3. 플러그인별 상세 소개

### 3-1. moai-core — AI 비서 (필수 설치)

**무엇을 하나요?** 사용자의 자연어 요청을 분석해 알맞은 스킬로 자동 연결하는 "AI 비서"입니다. 모든 플러그인의 중심 역할을 합니다.

| 기능 | 설명 |
|------|------|
| `/project init` | 프로젝트 초기화 — 어떤 일을 할지 인터뷰 후 맞춤 설정 생성 |
| AI 슬롭 검수 | 모든 텍스트 결과물에서 AI 특유의 어색한 표현을 자동 교정 |
| 자동 라우팅 | "상세페이지 만들어줘" → commerce 스킬 자동 실행 |

> **왜 필요한가요?** moai-core 없이는 다른 플러그인이 작동하지 않습니다. 가장 먼저 설치하세요.

### 3-2. moai-commerce — 한국 이커머스 풀세트 (22개 스킬)

**무엇을 하나요?** 상품 등록부터 광고 최적화, 매출 분석까지 이커머스 전 과정을 자동화합니다. 정해준 강사의 실전 노하우(쿠팡 매출 9배 비법)가 반영되어 있습니다.

#### Day 2 핵심 — 상품 기획 6도구 (V6)

| 스킬 | 하는 일 | 결과물 예시 |
|------|---------|------------|
| **시장조사** (`commerce-market-research`) | 시장 규모, 경쟁사, 포지셔닝 5축 분석 | "프리미엄 유기농 간식 시장은 연 2,400억, 경쟁 강도 중간" |
| **고객 페르소나** (`commerce-jtbd-persona`) | 고객이 '어떤 일'을 하려고 하는지(JTBD) 분석 | "30대 직장인 엄마, 아이 간식 고민 해소" 페르소나 카드 |
| **상품명 작성** (`commerce-product-naming`) | 검색 최적화 상품명 3안 + 검증 | "[유기농] 아이간식 스낵세트 베이비오 12종 / 선물용" |
| **채널별 메시지** (`commerce-channel-message`) | 쿠팡/네이버/인스타 채널에 맞는 멘트 | 쿠팡: "검색어 최적화형", 인스타: "감성 스토리형" |
| **통합 전략** (`commerce-integrated-strategy`) | 채널 믹스 + 가격 + 프로모션 캘린더 | "1개월: 쿠팡 유입 → 2개월: 네이버 정착 → 3개월: 인스타 확산" |
| **상세페이지 카피** (`detail-page-copy`) | 13섹션 감정 여정 카피 (Hero → Pain → CTA) | "우리 아이 첫 간식, 엄마는 3번 확인합니다" 오프닝 카피 |

#### 상세페이지 제작

| 스킬 | 하는 일 |
|------|---------|
| **상세페이지 이미지** (`detail-page-image`) | 섹션별 이미지 프롬프트 자동 생성 → AI 이미지 → 세로 롱 이미지 합성 |
| **상품 사진 브리프** (`product-photo-brief`) | 촬영해야 할 컷 13장 + 앵글·조명 가이드 |

#### 채널별 마켓 가이드

| 스킬 | 채널 | 주요 내용 |
|------|------|----------|
| `marketplace-coupang` | 쿠팡 | 로켓배송 기준, 검색 키워드 전략, 금지 문구 |
| `marketplace-naver` | 네이버 | 스마트스토어 + 11번가/G마켓/옥션 |
| `marketplace-d2c` | 자사몰 | 카페24·아임웹·메이크샵 운영 가이드 |
| `marketplace-crowdfunding` | 크라우드펀딩 | 와디즈·텀블벅 기획·심사 |
| `marketplace-curation` | 큐레이션 | 카카오 메이커스·무신사·29CM 입점 |

#### 광고 & 매출 최적화

| 스킬 | 하는 일 |
|------|---------|
| **쿠팡 광고 최적화** (`coupang-ad-optimizer`) | 3가지 캠페인 유형 자동 분류, 검색/비검색 분리, 엔드 ROAS 계산, 자동 규칙 3종 |
| **마진·ROAS 계산기** (`commerce-margin-calculator`) | 채널별 수수료 자동 반영, 손익분기 광고비 산출 |
| **자동화 진단** (`commerce-automation-audit`) | 6대 영역 진단 → 우선순위 점수 → 3 Phase 로드맵 |
| **식약처 안전** (`mfds-safety`) | 건강기능식품 인정 여부, 검사 부적합, 회수 정보 확인 |

#### 매장 운영

### 3-3. moai-media — AI 미디어 스튜디오 (13개 스킬)

**무엇을 하나요?** 글을 입력하면 AI가 이미지, 영상, 음성을 만들어줍니다. 상세페이지 소재, 광고 영상, SNS 콘텐츠 제작에 직접 활용합니다.

#### Day 3 핵심 — AI 미디어 6도구

| 스킬 | 하는 일 | 결과물 |
|------|---------|--------|
| **무드보드** (`media-moodboard`) | 색상 팔레트·톤·레퍼런스 자동 생성 | 상품에 맞는 비주얼 방향성 |
| **GPT Image 2 프롬프트 빌더** (`media-gpt-image2-builder`) | 자연어 → 8단계 프롬프트 자동 변환 | "화장품 샘플 이미지 만들어줘" → 고품질 이미지 |
| **모델 라우터** (`media-model-router`) | 용도에 맞는 AI 모델 자동 선택 | 이미지용? 영상용? 립싱크용? 자동 판단 |
| **채널 광고 패키저** (`media-channel-ad-packager`) | 메타·네이버·카카오 광고 규격 자동 변환 | 1장의 이미지 → 3개 채널 규격 동시 생성 |
| **AI 표기** (`media-ai-disclosure`) | 광고심의 "AI 생성" 메타데이터·워터마크 자동 부착 | 심의 대응 파일 |
| **캔바 매직 레이어** (`media-canva-magic-layer`) | 시즌별 디자인 재사용 가이드 | 명절·세일 등 시즌 소스 재활용 |

#### 기본 이미지·영상 생성

| 스킬 | 하는 일 |
|------|---------|
| **나노바나나** (`nano-banana`) | Gemini 기반 한국어 텍스트가 잘 들어가는 이미지 생성 |
| **범용 이미지** (`image-gen`) | Nano Banana / Flux / Recraft / Imagen 자동 선택 |
| **범용 영상** (`video-gen`) | Kling·Hailuo·Luma·Pika·Veo 통합 — 숏폼/롱폼 |
| **음성/TTS** (`audio-gen`) | ElevenLabs·Suno·MiniMax — 더빙·BGM·효과음 |
| **토킹헤드** (`speech-video`) | 정지 이미지 + 음성 → 입모양 동기화 영상 |
| **캐릭터 관리** (`character-mgmt`) | 일관된 캐릭터·아바타 시리즈 생성 |
| **fal.ai 게이트웨이** (`fal-gateway`) | 1000개 이상 AI 모델 통합 접근 |

> **어떤 API 키가 필요한가요?**
> - `GEMINI_API_KEY` — 나노바나나 이미지 생성 (Google AI Studio에서 무료 발급)
> - `FAL_KEY` — fal.ai 범용 게이트웨이 (fal.ai에서 발급)
> - `ELEVENLABS_API_KEY` — 음성 생성 (ElevenLabs에서 발급)
> - `HIGGSFIELD_API_KEY` — 토킹헤드 영상 (Higgsfield에서 발급)

### 3-4. moai-marketing — 마케팅 (10개 스킬)

**무엇을 하나요?** 광고 기획, SNS 콘텐츠, 성과 분석까지 마케팅 전 과정을 지원합니다. 광고 심리학 13장이 통합되어 있어 효과적인 카피를 자동 생성합니다.

#### 캠프 핵심 스킬

| 스킬 | 하는 일 |
|------|---------|
| **SNS 콘텐츠** (`sns-content`) | 네이버 블로그·인스타·카카오 채널 최적화 콘텐츠. 채널별 심리 매트릭스 + 메타 학습 기간 반영 |
| **캠페인 플래너** (`campaign-planner`) | 광고 심리학 기반 기획 — 6가지 방아쇠·8가지 편향·PAS·후크 6종·30초 영상 스크립트 |
| **랜딩 진단** (`landing-page-conversion-audit`) | 랜딩페이지 6섹션 진단 — CTR/CVR 분기, 불안해소 문구, 메시지 일치 |
| **픽셀 검증** (`pixel-audit`) | 메타·구글 픽셀, CAPI, Lookalike 씨앗 품질 진단 |
| **메타 광고 분석** (`meta-ads-analyzer`) | .xlsx 보고서 → 9개 모듈 분석 + 4D 교차분석 + 3가지 사용자 그룹 톤 + 4가지 출력 형식 |

### 3-5. moai-content — 콘텐츠 제작 (11개 스킬)

**무엇을 하나요?** 랜딩페이지, 카드뉴스, 블로그 글, 카피 등 모든 텍스트 콘텐츠를 생성합니다. 모든 결과물은 AI 특유의 어색한 표현을 자동 교정합니다.

| **바른한글 맞춤법** (`korean-spell-check`) | 부산대 바른한글 기반 맞춤법·띄어쓰기 검수 |
| `blog` | SEO 최적화 블로그 글 작성 |
| `newsletter` | 뉴스레터 기획~발행 |
| `media-production` | 유튜브 스크립트, 영상 기획 |
| `product-detail` | 전환율 극대화 상세페이지 |
| `social-media` | 멀티 채널 소셜 콘텐츠 |

> **품질 보증 체인**: 모든 텍스트 산출물은 자동으로 AI 슬롯 검수 → 한국어 정밀 윤문 → 맞춤법 검수 순으로 후처리됩니다.

## 4. 3일 캠프 일별 도구 사용 흐름

### Day 1 — 셋업 & 매장 운영

```
┌─────────────────────────────────────────────────┐
│  /project init → 워크플로우 인터뷰 → CLAUDE.md   │
│                                                 │
│  MCP 커넥터 연결 (mcp-connector-setup)           │
│  ├── Google Drive (상품 이미지)                   │
│  ├── Notion (작업 관리)                           │
│  └── OpenAI / Higgsfield (미디어 API)            │
│                                                 │
│  매장 운영 데이터 확인                            │
│  ├── commerce-morning-brief (오늘 할 일)          │
│  └── commerce-order-summary (주문 현황)           │
└─────────────────────────────────────────────────┘
```

### Day 2 — 상품 기획 & 상세페이지 (V6 6도구)

```
┌─────────────────────────────────────────────────┐
│  V6 워크플로우 (정해준 강사 7교시 구조)           │
│                                                 │
│  ① 시장조사 (commerce-market-research)           │
│     → 시장 규모, 경쟁사, 포지셔닝 분석            │
│                                                 │
│  ② 고객 페르소나 (commerce-jtbd-persona)          │
│     → JTBD + 타겟 온도 + 심리적 필요              │
│                                                 │
│  ③ 상품명 (commerce-product-naming)              │
│     → 검색 최적화 3안 + 금지 키워드 검증          │
│                                                 │
│  ④ 채널 메시지 (commerce-channel-message)        │
│     → 쿠팡/네이버/인스타 채널별 멘트              │
│                                                 │
│  ⑤ 통합 전략 (commerce-integrated-strategy)      │
│     → 채널 믹스 + 프로모션 캘린더                │
│                                                 │
│  ⑥ 상세페이지 카피 (detail-page-copy)            │
│     → 13섹션 감정 여정 카피 + 페르소나 카피 2세트  │
│                                                 │
│  보너스: 상세페이지 이미지 (detail-page-image)     │
│     → AI 이미지 생성 → 세로 롱 이미지 합성        │
```

## 6. 설치 방법 (3분)

### Step 1: 마켓플레이스 추가

1. Claude Cowork 좌측 사이드바 > **사용자 지정(Customize)** 클릭
2. **개인 플러그인** 영역에서 **+** 버튼 클릭
3. **플러그인 추가 > 마켓플레이스 추가** 선택
4. URL 입력: `modu-ai/cowork-plugins` → **동기화** 클릭

### Step 2: 플러그인 설치

1. **moai-core** 를 가장 먼저 설치 (+ 버튼)
2. 이어서 아래 플러그인 설치:
   - `moai-commerce`
   - `moai-media`
   - `moai-marketing`
   - `moai-content`
   - `moai-office`

### Step 3: 프로젝트 초기화

채팅창에 입력:

```
/project init
```

1. 분야 선택 → **비즈니스/이커머스**
2. 플러그인 선택 → 위 6개 플러그인
3. API 키 등록 (선택)
4. 완료 — 이후 자연어로 바로 사용 가능

## 7. 자주 묻는 질문

### Q: 코딩 지식이 필요한가요?

**아닙니다.** 모든 기능을 한국어 대화로 사용합니다. "상세페이지 만들어줘", "광고 영상 만들어줘"처럼 말하면 됩니다.

### Q: API 키는 필수인가요?

**아닙니다.** 텍스트 생성(상세페이지 카피, SNS 콘텐츠 등)은 API 키 없이도 작동합니다. **이미지·영상·음성** 생성에만 각 서비스의 API 키가 필요하며, Google AI Studio 키는 무료로 발급 가능합니다.

### Q: 캠프 전에 미리 설치해야 하나요?

**네, 권장합니다.** 캠프 시작 전 Step 1~3까지 완료해 오시면 Day 1부터 바로 실습에 들어갈 수 있습니다. 설치에 3분 정도 소요됩니다.

### Q: 캠프 후에도 계속 쓸 수 있나요?

**네, 무제한입니다.** moai-cowork은 MIT 라이선스 오픈소스로, 설치 후 계속 무료로 사용할 수 있습니다. 새 상품 등록, 광고 소재 제작, 매출 분석 등 일상적인 셀러 업무에 바로 활용하세요.

### Q: 다른 마켓(11번가, G마켓)도 지원하나요?

**네.** 쿠팡, 네이버 스마트스토어 외에도 11번가, G마켓, 옥션, 카페24, 아임웹, 메이크샵, 와디즈, 텀블벅, 카카오 메이커스, 무신사, 29CM까지 가이드가 포함되어 있습니다.

### Q: 식약처 관련 검수도 되나요?

**네.** `mfds-safety` 스킬로 건강기능식품 인증 여부, 검사 부적합 이력, 회수 정보를 확인할 수 있습니다. 건강기능식품·영양제 셀러에게 특히 유용합니다.
