# 데일리 브리핑 (Daily Briefing)

매일 아침 업무 시작 전 5분 만에 비즈니스 현황을 파악할 수 있는 구조화된 브리핑을 생성합니다. 국내외 업계 뉴스, 경쟁사 동향, 주요 지표, 오늘의 액션 아이템을 하나의 리포트로 통합합니다.

## 개요

이 스킬은 사용자 업종·관심 주제에 맞게 브리핑 항목을 맞춤 설정하여 구조화된 일일/주간 브리핑을 생성합니다. 업계 주요 뉴스, 경쟁사 모니터링, 규제·정책 업데이트, 시장 지표, 오늘의 액션 아이템을 포함합니다.

## 트리거 키워드

- 일일 브리핑, 데일리 리포트, 오늘의 이슈, 오늘 뉴스
- 아침 브리핑, 비즈니스 동향, 시장 동향, 뉴스 요약
- 경쟁사 모니터링, 주간 브리핑, 위클리 리포트

## 워크플로우

### 1단계: 브리핑 범위 설정

- 대상 업종 및 키워드 확인 (`.moai/config.json`의 `industry` 설정 우선 사용)
- 브리핑 기간 설정 (일간/주간/월간)
- 모니터링 경쟁사 목록 확인

### 2단계: 정보 수집 및 분류

다음 섹션으로 구조화하여 수집:

**국내 업계 뉴스**
- 네이버 뉴스, 조선일보 경제, 한국경제, 파이낸셜뉴스
- 업종별 주요 매체 (IT: 블로터, ZDNet; 스타트업: 벤처스퀘어)

**글로벌 동향**
- WSJ, TechCrunch, Bloomberg 주요 헤드라인 요약
- 글로벌 트렌드 및 시사점

**경쟁사 모니터링**
- 경쟁사 채용 공고, 보도자료, 제품 업데이트
- SNS 활동, 마케팅 캠페인

**규제·정책 변화**
- 금융위, 공정거래위, 중기부, 과기정통부 공지
- 정책 브리핑(korea.kr)

**시장 지표**
- 코스피/코스닥, 환율, 유가, 금리 동향

### 3단계: 브리핑 리포트 생성

아래 표준 포맷으로 마크다운 리포트 생성:

```markdown
# [날짜] 데일리 브리핁

## 헤드라인 요약 (3줄 핵심)
1. ...
2. ...
3. ...

## 업계 주요 뉴스
| 제목 | 출처 | 시사점 |

## 경쟁사 동향

## 규제/정책 업데이트

## 시장 지표

## 오늘의 액션 아이템
- [ ] ...
```

### 4단계: 사용자 검토 및 저장

- 리포트를 사용자에게 제시
- 피드백 반영 후 `.moai/briefings/YYYYMMDD.md`에 저장 (옵션)

## 사용 예시

### 예시 1: SaaS 업계 데일리 브리핑

**입력:**
```
업종: SaaS
기간: 오늘
경쟁사: 워크립, 리브
```

**출력:**
```
# 2026년 4월 30일 데일리 브리핑

## 헤드라인 요약 (3줄 핵심)
1. 네이버, HR 테크 SaaS '워크립'에 500억원 추가 투자 발표
2. 정부, 중소기업 AI 도입 지원 확대 (내년부터 30% 세제 혜택)
3. 리브(채용플랫폼), 일본 시장 진출 발표

## 업계 주요 뉴스
| 제목 | 출처 | 시사점 |
|------|------|--------|
| 네이버, 워크립 500억원 추가 투자 | 한국경제 | HR 테크 시장 과열, 경쟁 심화 예상 |
| 정부, 중소기업 AI 도입 세제 혜택 | 정책브리핑 | B2B SaaS 타겟 기업의 구매력 증대 |
| 카카오엔터프라이즈, AI 협업툴 출시 예정 | 전자신문 | 협업툴 시장 신규 진입, 경쟁 확대 |

## 경쟁사 동향
**워크립**
- 대기업 타겟 '워크립 엔터프라이즈' 베타 서비스 시작
- 채용: AI 엔지니어 10명 급증 (연봉 1억원+)

**리브**
- 일본 'Recruit'와 제휴 발표 (일본 시장 진출)
- 리브 오픈 API 개발자 컨퍼런스 예정

## 규제/정책 업데이트
- **개인정보보호법**: AI 학습 데이터 위임 동의 가이드라인 발표 (행안부)
- **플랫폼 사업법**: SaaS 플랫폼은 적용 제외 (금융위)

## 시장 지표
- 코스닥: 875.42 (+0.8%)
- 원/달러 환율: 1,352원 (+0.3%)
- 기준금리: 3.50% (동결)

## 오늘의 액션 아이템
- [ ] 워크립 엔터프라이즈 베타 신청 후 기능 분석
- [ ] 정부 AI 지원 세제 혜택 요건 확인 및 활용 방안 검토
- [ ] 리브 일본 진출 관련 우리 서비스의 동남아 진입 가능성 검토
```

### 예시 2: 주간 핀테크 시장 동향

**입력:**
```
업종: 핀테크
기간: 이번 주
경쟁사: 카카오페이, 토스
```

**출력:**
```
# 2026년 4월 4주 주간 브리핑 (핀테크)

## 헤드라인 요약 (3줄 핵심)
1. 금융위, 마이데이터 사업 3단계 확대 발표 (은행권 반발)
2. 카카오페이, 미국 하와이서 QR 결제 서비스 시작
3. 토스, 청년층 타겟 '토스 체크카드' 출시 (연회비 무료)

## 업계 주요 뉴스
| 제목 | 출처 | 시사점 |
|------|------|--------|
| 금융위, 마이데이터 3단계 확대 | 파이낸셜뉴스 | 데이터 활용 범위 확대, 핀테크 기회 |
| 카카오페이 하와이 QR 결제 | TechCrunch | 글로벌 진출 가속화, 해외 결제 경쟁 |
| 토스 체크카드 출시 | 이데일리 | 카드사 직접 진입, 수익 다변화 |
| 신한은행, 카카오와 제휴 강화 | 머니투데이 | 빅텍-은행 제휴 확대 추세 |

## 경쟁사 동향
**카카오페이**
- 하와이에서 QR 결제 서비스 시작 (첫 미국 진출)
- 카카오페이 인증서 전 국민은행 계좌 연동 가능
- 채용: 글로벌 사업팀 20명 (미국, 일본 전문가)

**토스**
- '토스 체크카드' 출시 (연회비 무료, 청년층 타겟)
- 토스 증권 MT(주문) 거래량 국내 1위 달성
- '토스 3천만 유저' 달성 기념 캠페인

## 규제/정책 업데이트
- **마이데이터 3단계**: 신용정보, 부동산, 보험 데이터 추가 (내년 상반기)
- **전자금융거래법**: 핀테크 사업자 보호 강화 (금융위)
- **가상자산 규제**: 은행권 가상자산 예치 계좌 허용 방안 검토

## 시장 지표
- 코스닥: 875.42 (+1.2% 주간)
- 비트코인: 8,500만원 (+5.3% 주간)
- 기준금리: 3.50% (동결)

## 오늘의 액션 아이템
- [ ] 마이데이터 3단계 확대 내용 분석 및 우리 서비스 반영 방안 검토
- [ ] 카카오페이 글로벌 진출 사례 벤치마킹 (동남아 시장)
- [ ] 토스 체크카드 성능 및 마케팅 전략 분석
```

## 출력 형식

### 표준 섹션 구조

1. **헤드라인 요약**: 3줄 핵심 뉴스
2. **업계 주요 뉴스**: 제목, 출처, 시사점 테이블
3. **경쟁사 동향**: 주요 활동, 채용, 제품 업데이트
4. **규제/정책 업데이트**: 정부 부처 공지, 법안
5. **시장 지표**: 코스피/코스닥, 환율, 금리
6. **오늘의 액션 아이템**: 체크리스트

### 맞춤 설정

`.moai/config.json`에 다음 설정 추가:
```json
{
  "industry": "SaaS",
  "competitors": ["워크립", "리브"],
  "keywords": ["HR 테크", "협업툴", "AI"],
  "news_sources": ["블로터", "ZDNet", "한국경제"]
}
```

## 주의사항

### 정보 소스 신뢰성

- **경제/비즈니스**: 한국경제(hankyung.com), 매일경제(mk.co.kr), 파이낸셜뉴스(fnnews.com)
- **스타트업/테크**: 블로터(bloter.net), 벤처스퀘어(venturesquare.net), 이코노미스트
- **정책/규제**: 정책브리핑(korea.kr), 중기부 공식 채널
- **금융 지표**: 네이버 금융, KRX 한국거래소, 한국은행 ECOS

### 브리핑 자동화 팁

- `.moai/config.json`에 `industry`, `competitors`, `keywords` 설정 시 맞춤 브리핑 자동화
- 사용자가 "매일 아침 브리핑" 요청 시 이전 브리핑과 비교하여 변화 포인트를 강조
- TAM·SAM·SOM 기반 시장 동향 분석 가능 (market-analyst 스킬과 연동)

### TAM·SAM·SOM 시장 규모 분석 (투자자 브리핑 모드)

사업계획서 또는 IR 브리핑 요청 시 다음 공공 데이터 소스 활용:
- 통계청 KOSIS(kosis.kr)
- 한국무역협회 K-stat(stat.kita.net)
- 중소벤처기업부 K-스타트업(k-startup.go.kr) 통계

## 관련 스킬

- **market-analyst**: 심층 시장 분석이나 TAM/SAM/SOM 산출
- **investor-relations**: 투자자 보고서나 IR 덱 작성
- **consulting-brief**: 경쟁사 분석 포함 컨설팅 제안서
- **ai-slop-reviewer**: 생성된 브리핑 AI 패턴 검수 (필수)
